## Running

 - #### Clone this repository  

```bash
    $ git clone https://github.com/mateus-prog/app_b43fdd98.git
```

- #### Enter the backend folder and rename .env.example to .env.
```bash
    $ cp .env.example .env
```

- #### Access

```bash
    $ http://127.0.0.1:8000/api/
```
